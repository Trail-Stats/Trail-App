Comment utiliser ces analyses ?

Ces analyses vous permettent de :

  • Comprendre la difficulté de chaque section du parcours
  • Identifier les segments qui demandent le plus de technicité
  • Repérer les portions où se joue la course
  • Comparer les performances des coureurs sur des sections spécifiques

Sélectionnez une course dans le menu déroulant pour filtrer les analyses.
